A3 Technology test task
=======

Requirements
----------------
* iOS 10.0+
* Xcode 10.0+
* Swift 4.2+

Installation
----------------

```
git clone https://github.com/m3g0byt3/a3-technology-test-task.git\
 && cd a3-technology-test-task\
  && open A3TestTask.xcodeproj
```
